## Windows-Defender-Disabler

#### *Clone or Download this Repository 

#### *You can disable your AV by executing this BAT file. ``Run it as administrator``.

#### *It will disable AntiVirus and will help you to execute tools like openbullet and many more.

#### *Use this only on a Virtual Machine.
